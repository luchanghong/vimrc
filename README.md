## Powerline

- 安装`powerline status`

```
pip install powerline-status
```

- 配置`.zshrc`

添加一行

```
. /Users/lch/.vim/bundle/powerline/powerline/bindings/zsh/powerline.zsh
```

- 安装powerline字体

```
shell> git clone https://github.com/powerline/fonts.git
shell> cd fonts
shell> ./install.sh
```

- 设置`iTerm`终端字体

`Preferences`->`Profiles`->`Regular Font/Non-ASCII Font`
